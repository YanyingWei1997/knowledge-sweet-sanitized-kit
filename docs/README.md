## 根目录保留什么

根目录只保留两类内容：

- 运行文件：队列、状态、日志、脚本
- 入口文件：本 `README.md`

## 不再直接放根目录的内容

以下已迁出根目录：

- 已归档知识说明稿 → `知识库/`
- 旧兼容跳转文件 → `兼容跳转/`
- 历史副本 → `归档/`
