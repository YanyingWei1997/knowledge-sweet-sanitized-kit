## 核心功能
- 监听指定会话消息
- 自动识别内容类型
- 归类到正确的 tag/项目/年/月
- 生成预览并确认
- 写入Markdown文件

## 当前落地结构
- 运行层：`workspace/knowledge-sweet/`
- 知识层：`workspace/knowledge-sweet/知识库/tag/子类/项目/年/月/`

## 说明
- 队列、日志、状态、调度脚本保留在根目录
- 长期知识内容归档到 `知识库/`
- Obsidian 同步时优先同步 `知识库/`
- tag 使用固定白名单
- 同项目迭代以最新年月目录为正式保存位置
- 任何新内容入库都必须先完成 `tag -> 项目 -> 年/月` 分类
- 若无对应项目目录，先创建项目目录和 `LATEST.md`
- Prompt 或其他可复制内容，正文必须使用代码框保存
